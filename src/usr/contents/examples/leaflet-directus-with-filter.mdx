### Leaflet example with OSM basemap and a vector layer sourced form Directus API with filter

Only Hellenistic and Roman sites are being displayed

<div className="row mt-3 mb-5 border-top border-bottom py-3">
  <div className="col-6">
    <MapLeaflet
      height="400px"
      baseLayers={["OSM"]}>

      <VectorLayer
        source={{
          dTable: "scms_ksa",
          geoField: "geometry",
          dQueryString: "filter[Hellenistic_Roman][_eq]=true"
        }}
        name="Hellenistic and Roman Sites"
        
        checked={true}
        fitToContent={true}
        popupTemplate="<p><strong>${Item_Label}</strong>"
      />

    </MapLeaflet>
